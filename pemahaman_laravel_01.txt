.env berguna untuk mempermudah kita dalam konfigurasi pada projek kita, karena env berisi pengaturan-pengaturan pada project laravel, contohnya seperti konfigurasi database. .env juga  digunakan sebagai file informasi bagi semua tim untuk mengetahui kunci dan nilai apa yang mungkin diperlukan. 

Direktori config berguna untuk menyimpan beberapa file yang penting untuk menjalankan konfigurasi aplikasi laravel.
